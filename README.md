# Obsidian Customize Icons

Folder-based icon assignment with quality score and connectivity coloring for Obsidian.

## Features
- **Folder-based icons**: Assign an icon to a folder — all files inside inherit it
- **Subfolder overrides**: Subfolders can have their own icon
- **Visual icon picker**: Browse and search all available icons
- **Quality score coloring**: Notes with a `Quality score` frontmatter field get tinted icons
- **Connectivity coloring**: Densely-linked notes get a distinct color (amber by default)
- **Bundled icons**: Icons ship with the plugin for cross-machine sync via BRAT
- **Tab bar icons**: Show file icons in tab headers (toggleable)
- **Title icons**: Show icon above the note title (toggleable)
- **Link icons**: Show icons in internal links (toggleable)

## Settings
- Toggle icons in tabs, title, links, editor
- Color pickers for default, quality exists, quality high, connectivity
- Connectivity threshold with penalty folders
- Per-folder icon assignment with visual picker

## Install via BRAT
1. Install the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat)
2. Add beta plugin: `borthwick/obsidian-customize-icons`
